# 🗣️Ligatures in Langauges
Using an open source model for a font-based ligature detection tool.

### 📋Aims of App
- Allow users input a string of text that uses a ligature based language and gets a translation back
  
### 👩‍💻Technical Details

- React
- TypeScript
- Shadcn/UI
- Next.js
- Supabase
- Hugging Face
