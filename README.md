# 🗣️Ligatures in Langauges
Using an open source model for a font-based ligature detection tool.

### 📋Aims of App
- Allow users input a string of text that uses a speical ligature based language and get a translation back
  
### 👩‍💻Technical Details

- React
- TypeScript
- Shadcn/UI
- Next.js
- Python
- Hugging Face
